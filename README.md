## Modules List

### 1. Flutter Flavors 
Flavors is a generic term used for different versions of an app such as 'dev', 'staging', and 'prod'. when we build Market-ready app, to ensure that end-user should get error and bug-free product, we create differnt flavors of an app. The above mentioned flavors are the industry standard flavors.
